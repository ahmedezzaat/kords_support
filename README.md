# Customer Support Portal

A premium SaaS Administrative tool designed for tracking client tickets, managing team roles, and integrating with external developer services.

![Portal Preview](https://github.com/ahmedezzaat/kords_support/raw/main/preview.png)

## 🚀 Live Demo
Experience the dashboard live: [https://ahmedezzaat.github.io/kords_support/](https://ahmedezzaat.github.io/kords_support/)

## ✨ Key Features
- **📊 Real-time Dashboard**: Overview of companies, contacts, and active tickets.
- **🏢 Company Management**: Profile management for clients with slug-based identification.
- **🎫 Advanced Ticketing**: 
  - Priority-based classification (Low, Medium, High, Urgent).
  - Categorization (Bug, New Feature).
  - Status tracking (Open, In Progress, Resolved).
  - File attachments support.
- **👨‍💻 Team Collaboration**: Role-based access (Admin, Support Agent, Manager, Viewer).
- **🔗 External Integrations**: 
  - **GitHub**: Push internal tickets directly to a GitHub repository as issues.
  - **EmailJS**: Automated email notifications for ticket creation and updates.

## 🛠 Tech Stack
- **Frontend**: Vanilla HTML5, CSS3, Javascript.
- **UI Architecture**: Glassmorphism design with responsive grid layouts.
- **Iconography**: Remix Icons.
- **Typography**: Inter (Google Fonts).
- **Automation**: GitHub Actions (CI/CD).

## 💻 Local Development
Simply open `index.html` in any modern browser to start. The application uses `localStorage` to persist data locally.
